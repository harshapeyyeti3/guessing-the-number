# guessing-the-number
A clean and structured implementation of a two-player 3-digit guessing game in C.

Each player sets a hidden 3-digit secret number, and players alternate guessing each other's secret.
For each guess, the program reports:

Correct-position matches (right digit, right place)

Wrong-position matches (right digit, wrong place)

The match-checking logic avoids double counting and handles repeated digits correctly.

User input validation

Array-based match checking

Clear function separation and game loop logic
